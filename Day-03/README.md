# Day 03 - String Warrior

Learn string manipulation through mini-projects and games.

### Password Strength Checker
Checks whether a password is strong based on:
- Length
- Uppercase letters
- Lowercase letters
- Numbers
- Special characters

### String Challenges
- Palindrome Checker
- Vowel Counter
- Reverse String
- Character Frequency Counter

### Hangman Game
Built a word guessing game using loops and string operations.
